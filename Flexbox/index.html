¿Qué es Flexbox y cómo funciona?

Flexbox es una modelo de sieño WEB utilizado en CSS para ordenar o poner en cierta disposición los elementos de una página web. Esto de manera flexible. Siendo el sucesor de propiedades como float.

Más adelante vamos a ver las propiedades más importantes utilizados para Flexbox. Primero que nada debemos saber que para utilizar este modelo necesitamos de un objeto padre, al que le daremos la propiedad de display: flex; así como objetos hijos los cuales se distribuirán dentro del objeto padre. A continuación podemos ver la estructura básica de un archivo HTML, en el que son visibles los objetos padre e hijos.

Flex-wrap

Esta propiedad es muy útil para hacer que CSS respete el ancho de los objetos hijos. En este caso vamos a utilizar como ejemplo un objeto padre con seis hijos, con las siguientes dimensiones: 

.hijo {
    width: 70px;
    height: 30px;
}

Hay que recordar que estas son las dimensiones en un objeto padre pequeño, como los que hemos visto arriba. Ahora bien, si dejamos los seis objetos hijos con la propiedad flexbox dentro del padre, por defecto estos se distribuirán a lo largo de una línea, y por lo tanto perderan ancho. A menos que nosotros le digamos a CSS que queremos que respete de forma explicita ese ancho, poniendo los objetos que no caben en una línea en la siguiente.

flex-wrap: nowrap;

Justify-content

Otra propiedad interesante es Justify-content . Con ella podemos indicarle al CSS como va a justificar el contenido dentro del objeto padre. Esta propiedad se parece bastante al atributo text-align , pero orientada a todos los objetos del objeto padre (contendedores, imagenes, texto, etc.).

Nos ayuda a alinear el contenido horizontalmente

Align-items

Align-items es una propiedad que sirve para determinar la altura a la que se encontrarán los hijos dentro del contenedor al que le apliquemos esta propiedad. No se utiliza para determinar la altura en píxeles, sino la altura relativa al contenedor. Por ejemplo: el inicio, final o centro del contenedor.
